# BANKING APP
banking app with a finance management dashboard that connects multiple bank accounts, displays real-time transactions, and allows you to transfer money to other users on the platform using Next.js 14.
 
# TECH STACK USED:
Next.js 

TypeScript

Appwrite

Plaid

Dwolla

React Hook Form

Zod

TailwindCSS

Chart.js

ShadCN
